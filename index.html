Les languages

Programmation mobile

Dans le domaine de la programmation mobile, il existe de nombreux languages différents et tous avec des utilités différentes. Par exemple pour le développement d'applications android on peut utiliser Kotlin ou Java alors qu'on utiliserait Swift sur IOS. Tous sont équivalent mais différents, c'est pourquoi il peut être intéréssant de les apprendre.

Programmation Web Front-End

Le front-end c'est le côté client du site, il gère l'interaction et le style de la page. Parmis les nombreux languages utilisés on retrouve le trio bien connu : HTML, CSS, JS. Mais à partir du JS de nombreux languages ont été crées, c'est le cas du React améliorant les fonctionnalités et du jQuery faisant la même chose mais étant une bibliothèque.

Programmation Web Back-End

Le back-end est au contraire du front-end, invisible à l'utilisateur. L'un des principaux languages pour cela est le PHP, il n'est quasiment utilisé que pour le web. Le language le plus couramment utilisé dans la création de bases de données est le SQL qui permet le stockage d'informations en grande quantités.

Automatisations

Grâce à la programmation il est possible d'automatiser de nombreuses tâches. Tous les languages permettent à leur échelle d'automatiser des actions, par exemple grâce au javascript ou au python il est possible de créer des bots sur discord répondants aux attentes des utilisateurs et automatisant des actions. L'intelligence artifielle permet elle aussi d'automatiser des tâches, notamment le travail de tri dans les usines.

Machine Learning

Le machine learning est un sous domaine de l'intelligence artificielle. Il consiste à laisser des algorithmes découvrir et apprendre des patterns pour ensuite qu'ils puissent s'améliorer par eux même et devenir plus performants dans ces tâches. Le Python et le C++ excellent dans ces tâches, le python par sa facilité d'apprentissage et ses capacités hautes et le C++ par sa rapidité d'exécution et ses fonctionnalités.

Developpement de Jeux

La programmation de jeux est un domaine très spécifique dans lequel peux de languages sont suffisament adaptés aux requis. Le C++ encore une fois excelle dans cette branche de la programmation car avec sa rapidité d'exécution et sa compatibilité avec presque tous les moteurs de jeux il est n°1 sur le marché. Il sert principalement pour le développement de jeux sur consoles.
